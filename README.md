# Mini Trello Board (Vanilla JS)

A lightweight Trello-style board built with:
- HTML + CSS + JavaScript
- Lists + cards
- Add / rename / delete lists
- Add / edit (double click) / delete cards
- Drag & drop cards across lists
- Persists board state in `localStorage`

## How to run
Just open `index.html` in your browser.
